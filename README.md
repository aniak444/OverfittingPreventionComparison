# Overfitting Prevention in MLP Neural Networks

Comparison of overfitting prevention techniques in Multi-Layer Perceptron (MLP) neural networks.

**Course project** for **Computational Intelligence Methods**  
(Metody Inteligencji Obliczeniowej)

---

## About the project

The goal of this project is to:
- create MLP models that intentionally overfit on classification tasks,
- test and compare popular methods that help prevent overfitting,
- evaluate their effectiveness on real UCI benchmark datasets.
